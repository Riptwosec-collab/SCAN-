วิธีใช้งานแบบพร้อมส่งงาน

ทำตาม workflow นี้เพื่อให้ผล OCR ผิดน้อยลงและได้ไฟล์พร้อมใช้ต่อใน Ticket, Email, Report หรือเอกสารงาน

1. อัปโหลดเลือก รูปภาพ, PDF หรือ Batch แล้วใช้ Crop เฉพาะพื้นที่ข้อความถ้าภาพมีเมนู/พื้นหลังรบกวน
2. แปลงกดปุ่ม Scan ปุ่มเดียว ระบบจะลอง OCR หลายโหมดอัตโนมัติและเลือก candidate ที่มีคะแนนดีที่สุดให้เอง ไม่ต้องเลือก Preset เอง
3. ตรวจ Quality Gateดูเปอร์เซ็นต์, fixed words, คำเตือน และ OCR Candidates ก่อนนำผลไปใช้จริง
4. Exportส่งออก TXT, DOC, CSV, JSON หรือ PDF พร้อมหัวรายงาน วันที่ ชื่อไฟล์ และ confidence

เคล็ดลับใช้งาน

  • ระบบเลือกวิธีอ่านภาพให้อัตโนมัติทุกครั้งตามชนิดเอกสาร ไม่ต้องตั้งค่าก่อนสแกน
  • ใช้ปุ่ม Compare เพื่อเทียบ Raw OCR กับ Cleaned Output แบบสองคอลัมน์ก่อน export
  • ถ้าผล OCR ยังไม่แม่น ลอง Crop ให้เหลือเฉพาะข้อความ แล้วกด Scan ใหม่อีกครั้ง
  • เมื่อต้องใช้บ่อย สามารถติดตั้งเป็นแอปจาก browser ได้หาก browser รองรับ PWA

เกณฑ์ก่อนส่งงานจริง

  • Quality Gate ควรมากกว่า 85% สำหรับงานที่มีตัวเลข/URL/ชื่อเฉพาะสำคัญ
  • ถ้าคะแนนต่ำกว่า 70% ให้ Crop ใหม่ให้เหลือเฉพาะข้อความ หรือเลือก OCR Candidate อื่นก่อน Export
  • ตรวจเลขเอกสาร วันที่ ยอดเงิน URL อีเมล และชื่อระบบด้วยตาอีกครั้งเสมอ
  • เก็บ raw output และ cleaned output เป็น JSON เมื่อต้องการ audit ภายหลัง
