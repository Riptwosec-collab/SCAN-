วิธีใช้งานแบบพร้อมส่งงาน

ทำตาม workflow นี้เพื่อให้ผล OCR ผิดน้อยลงและได้ไฟล์พร้อมใช้ต่อใน Ticket, Email, Report หรือเอกสารงาน

1. อัปโหลดเลือก รูปภาพ, PDF หรือ Batch แล้วใช้ Crop เฉพาะพื้นที่ข้อความถ้าภาพมีเมนู/พื้นหลังรบกวน
2. เลือก Presetเลือก ใบกำกับภาษี, Ticket IT, Email แจ้งเตือน, เอกสารราชการ หรือ ตาราง/ฟอร์ม ให้ตรงกับต้นฉบับ
3. แปลงกดแปลง ระบบจะลอง OCR หลายโหมดและเลือก candidate ที่มีคะแนนดีที่สุด
4. ตรวจ Quality Gateดูเปอร์เซ็นต์, fixed words, คำเตือน และ OCR Candidates ก่อนนำผลไปใช้จริง
5. Exportส่งออก TXT, DOC, CSV, JSON หรือ PDF พร้อมหัวรายงาน วันที่ ชื่อไฟล์ และ confidence

Demo Sample ในหน้า Tool

  • กดปุ่ม ใบกำกับภาษี, Ticket IT, Email หรือ ราชการ เพื่อโหลด raw OCR จำลองแล้วดูผลก่อน/หลังทันที
  • กดปุ่ม ภาพ Mock เพื่อสร้างภาพตัวอย่างบน canvas แล้วทดสอบ OCR จากรูปจริงใน browser
  • ใช้ปุ่ม Compare เพื่อเทียบ Raw OCR กับ Cleaned Output แบบสองคอลัมน์ก่อน export
  • ถ้า OCR ไทยยังเพี้ยน ให้ลอง Engine: Auto ก่อน ถ้า browser รองรับ Native OCR ระบบจะเทียบผลกับ Tesseract ให้เอง
  • เมื่อต้องใช้บ่อย สามารถติดตั้งเป็นแอปจาก browser ได้หาก browser รองรับ PWA

เกณฑ์ก่อนส่งงานจริง

  • Quality Gate ควรมากกว่า 85% สำหรับงานที่มีตัวเลข/URL/ชื่อเฉพาะสำคัญ
  • ถ้าคะแนนต่ำกว่า 70% ให้ Crop ใหม่ เปิด Contrast หรือเลือก OCR Candidate อื่นก่อน Export
  • ตรวจเลขเอกสาร วันที่ ยอดเงิน URL อีเมล และชื่อระบบด้วยตาอีกครั้งเสมอ
  • เก็บ raw output และ cleaned output เป็น JSON เมื่อต้องการ audit ภายหลัง
