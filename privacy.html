Privacy Policy

นโยบายความเป็นส่วนตัว

RIPTWOSEC.SCAN เป็นเว็บ OCR แบบ Static Website สำหรับแปลงข้อความจากรูปภาพและ PDF โดยทำงานบน Browser เป็นหลัก

ไฟล์ที่เลือกใช้งาน

ไฟล์รูปภาพหรือ PDF จะถูกอ่านบน Browser ของผู้ใช้ผ่าน JavaScript เช่น Tesseract.js และ pdf.js ตัวเครื่องมือนี้ไม่อัปโหลดไฟล์เอกสารไปยัง server ของ RIPTWOSEC.SCAN

การประมวลผลข้อความ

การล้างคำผิด การรวมคำไทย และ dictionary ทำงานด้วยกฎภายในหน้าเว็บเป็นหลัก ไม่มีการส่งข้อความเอกสารไปยังบริการวิเคราะห์ข้อความภายนอกจากตัวเครื่องมือนี้

ประวัติ

ผลลัพธ์ OCR อาจถูกเก็บใน localStorage ของ Browser เพื่อให้เปิดดูย้อนหลังได้ ผู้ใช้สามารถล้างข้อมูลเว็บไซต์ใน Browser เพื่อลบประวัติ

การเก็บเงิน

เครื่องมือนี้ไม่มีระบบเก็บเงินหรือรหัสปลดล็อกในหน้าเว็บ ผู้ใช้ไม่ควรกรอกข้อมูลบัตรหรือข้อมูลการเงินลงในช่องใด ๆ ของเครื่องมือนี้

Library ภายนอก

เว็บนี้โหลด library และฟอนต์จาก CDN ภายนอกเพื่อให้เครื่องมือทำงานได้

คำแนะนำ

ควรระมัดระวังเมื่อนำเอกสารที่มีข้อมูลส่วนตัวหรือข้อมูลภายในองค์กรมาใช้งาน
