ไทย + อังกฤษ
เลือกภาษา OCR ให้เหมาะกับเอกสารไทยล้วน อังกฤษล้วน หรือปนกัน
Engine: Auto / Tesseract / Browser Native
Auto จะลอง Native OCR ถ้า browser รองรับ แล้วเทียบกับ Tesseract เพื่อเลือกผลที่ดีที่สุด
ข้อความล้วน / จัดเป็นเอกสาร / อีเมล / Ticket
เลือกรูปแบบจัดข้อความหลัง OCR ตามงานที่จะเอาไปใช้ต่อ
Preset: Auto / Document / IT Screenshot / Mobile / Table
ปรับวิธีเตรียมภาพให้เข้ากับต้นฉบับ เช่น เอกสารสแกน ภาพมือถือ หรือหน้าจอระบบ
Cleanup: Raw / Light / Balanced / Strict
กำหนดความแรงในการล้างคำเพี้ยน โดย Balanced เหมาะกับงานทั่วไป
PDF: แนวตั้ง / แนวนอน
กำหนดแนวเอกสารสำหรับ DOC/PDF export และ metadata ของไฟล์ที่ส่งออก
ลบอักษรแปลก / รวมคำไทยผิดช่องว่าง

Dictionary หลายสาย

รายการคำที่แก้ / ขยายภาพ / Contrast
